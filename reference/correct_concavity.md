# Correct concavity of Bellman values for the given weeks to have nice monotony for water values, used in `Grid_Matrix`

Correct concavity of Bellman values for the given weeks to have nice
monotony for water values, used in `Grid_Matrix`

## Usage

``` r
correct_concavity(df_value_node, weeks)
```

## Arguments

- df_value_node:

  DataFrame containing bellman values

- weeks:

  Weeks for which we want to correct concavity

## Value

vector of corrected Bellman values

# Convert water values to Antares format with high accuracy

This function converts water values generated by
[`Grid_Matrix()`](https://rte-antares-rpackage.github.io/antaresWaterValues/reference/Grid_Matrix.md)
to the format expected by Antares: a 365\*101 matrix, where the rows are
the 365 days of the year and the columns are round percentage values
ranging from 0 to 100 assessing the reservoir level. Use
[`antaresEditObject::writeWaterValues()`](https://rte-antares-rpackage.github.io/antaresEditObject/reference/writeWaterValues.html)
to write water values.

## Usage

``` r
to_Antares_Format_bis(data)
```

## Arguments

- data:

  A
  [`dplyr::tibble()`](https://tibble.tidyverse.org/reference/tibble.html)
  representing water values, output `aggregated_results` generated by
  [`Grid_Matrix()`](https://rte-antares-rpackage.github.io/antaresWaterValues/reference/Grid_Matrix.md).

## Value

A 365\*101 numeric matrix
